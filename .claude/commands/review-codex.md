# Codex Code Review

codex CLI を使用してコードレビューを実行します。

## 実行手順

1. 変更されたファイルを特定
2. codex にレビューを依頼
3. 結果をユーザーに報告

---

以下の手順で codex によるコードレビューを実行してください：

1. まず `git diff --name-only` で変更されたファイルを確認してください

2. 変更がある場合、以下のコマンドで codex にレビューを依頼してください：

```bash
codex --approval-mode full-auto "以下の git diff の変更内容をレビューしてください。問題点、改善点、セキュリティ上の懸念があれば指摘してください。日本語で回答してください。\n\n$(git diff)"
```

3. 変更がない場合は、ステージングされた変更を確認してください：

```bash
codex --approval-mode full-auto "以下の git diff の変更内容をレビューしてください。問題点、改善点、セキュリティ上の懸念があれば指摘してください。日本語で回答してください。\n\n$(git diff --staged)"
```

4. codex の出力結果をユーザーにわかりやすく要約して報告してください

**注意**: codex がインストールされていない場合は、`npm install -g @openai/codex` でインストールするようユーザーに案内してください。
