## 既知の問題

### 1. Sass Deprecation Warnings
- Bootstrap 5.3.3 が古いSass構文を使用
- 動作には影響なし、将来的にBootstrapのアップデートが必要

### 2. 運航状況API
- CORS制限により直接アクセスできない可能性
- プロキシサーバーまたはサーバーサイドでの取得が必要

### 3. oxc-parser警告
- 開発環境でのみ発生
- アプリケーションの動作には影響なし

## アクセス方法

1. 開発サーバーの起動:
   ```bash
   npm run dev
   ```

2. ブラウザでアクセス:
   - http://localhost:3030/

3. 各ページへのナビゲーション:
   - 時刻表: http://localhost:3030/timetable
   - 乗換案内: http://localhost:3030/transit
   - 運航状況: http://localhost:3030/status
