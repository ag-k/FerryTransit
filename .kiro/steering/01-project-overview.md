# プロジェクト概要

## FerryTransit について

FerryTransitは、島根県隠岐諸島のフェリー時刻表と航路情報システムです。このアプリケーションは各港間のフェリースケジュールを表示し、日本語と英語の両方をサポートしています。

## 現在の開発状況

**Phase 5.5 完了済み** - 管理画面実装まで完了しており、次はPhase 6（Capacitor統合・ネイティブアプリ化）が予定されています。

## 技術スタック

- **フレームワーク**: Nuxt 3.17.5 + Vue 3.5
- **言語**: TypeScript (strict mode)
- **スタイリング**: Tailwind CSS v4.1.10
- **状態管理**: Pinia 2.4.0
- **国際化**: @nuxtjs/i18n 9.6
- **認証**: Firebase Authentication
- **データベース**: Firebase Firestore
- **ストレージ**: Firebase Storage
- **テスト**: Vitest + Vue Test Utils
- **モバイル**: Capacitor 7.4.1

## プロジェクト構成

```
FerryTransit/
├── src/                   # Nuxt3版（メイン開発）
├── archive/               # AngularJS版（レガシー、参照のみ）
├── config/                # 設定ファイル
├── docs/                  # プロジェクトドキュメント
└── i18n/                  # 多言語対応
```

## 主要機能

### ユーザー向け機能
- 時刻表表示、乗換案内、運航状況確認
- 多言語対応（日本語・英語）
- 料金計算、お気に入り機能
- オフライン対応、ダークモード

### 管理者向け機能
- データ管理（時刻表、料金、アラート、お知らせ）
- 分析機能、公開機能
- Firebase統合による認証・権限管理

## 開発時の基本方針

1. **日本語での開発**: コミュニケーションは日本語で行う
2. **型安全性**: TypeScript strict modeを維持
3. **テスト駆動**: 新機能追加時は必ずユニットテストを作成
4. **国際化対応**: すべてのテキストはi18n経由で表示
5. **アクセシビリティ**: WCAG 2.1 AA準拠を目指す
6. **モバイルファースト**: レスポンシブデザインを重視
